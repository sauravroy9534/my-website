# Ritu Proposal Website

## Files
- index.html
- style.css
- script.js
- images/ (put your photos as `1.png`, `2.png`, `3.png`, ...)
- assets/ (put your song as `romantic.mp3` and background video as `love-loop.mp4`)
- assets/characters/ (graphic character SVGs used by proposal buttons)
- ANSWERS_PRIVATE.md (only for Saurav)

## How to use images
1. Put numbered PNG files in `images/`.
2. Quiz stages show `1.png` to `8.png`.
3. Final gallery auto-loads any available images from `1.png` to `40.png`.

## Run locally
Open `index.html` directly in browser.

## Premium Laptop Experience
- Full-screen cinematic entry screen.
- Looping romantic background video with dim dream effect.
- Floating kiss marks in background.
- Background music player (starts after first click).
- Music/video controls at top-right: pause/play + volume.
- Special timed music-name question: 15s limit, auto-skip with -1 negative mark on timeout.
- On every correct answer: \"Hooray\" + interactive blue hearts from bottom.
- Keyboard shortcuts:
  - `Enter`: continue flow (enter/start/next/open gallery)
  - `Esc`: pause/resume music

## Music setup
1. Copy your romantic song into `assets/`.
2. Rename it exactly to `romantic.mp3`.
3. Copy your short looping background video into `assets/`.
4. Rename it exactly to `love-loop.mp4`.
5. Open website and click `Enter The Experience` once to allow playback.

## Free hosting (GitHub Pages)
1. Create a GitHub repository (example: `ritu-proposal`).
2. Upload all files and folders from this project.
3. In GitHub repo: `Settings` -> `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main` and folder `/ (root)`.
6. Save and wait 1-2 minutes.
7. You will get a free URL like:
   `https://<your-username>.github.io/ritu-proposal/`

## Share via Bitly
1. Copy GitHub Pages URL.
2. Open bitly.com.
3. Paste URL and create short link.
4. Share short link with Ritu.
